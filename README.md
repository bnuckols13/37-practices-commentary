## Your writing

Put drafts and finished chapters in **`commentary/`**. The starter file [commentary/00-draft.md](commentary/00-draft.md) is a blank workspace; rename or split as you like (e.g. one file per verse or per session).

## Suggested workflow

1. Anchor each section to the **root text** (e.g. *Lojong* / Thogme Zangpo) and optionally note **timestamps** in the video.
2. Treat `source/*.txt` as **read-only reference**; keep corrections and interpretation in `commentary/`.
3. When you add more videos, drop transcripts in `source/` and link them here.
